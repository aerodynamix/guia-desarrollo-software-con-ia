# 18 - Estrategias de Monetización

> "El código es arte, pero el servidor se paga con dinero."
> No dejes la monetización para el final. Diséñala desde el inicio.

---

## 1. Publicidad (Ad-supported)

### Modelos
*   **CPM (Costo por Mil):** Te pagan por cada 1000 visualizaciones. Ideal para sitios con mucho tráfico (Noticias, Herramientas gratuitas).
*   **CPC (Costo por Click):** Te pagan si el usuario hace click. Ideal para nichos específicos (Seguros, Finanzas).

### Implementación Web (Google AdSense)
1.  **Bloques Automáticos:** Google decide dónde ponerlos. Fácil pero intrusivo.
2.  **Bloques Manuales (Recomendado):**
    *   **Header:** Banner 728x90. Alta visibilidad.
    *   **Sidebar:** Banner 300x250.
    *   **In-Article:** Entre párrafos de contenido. El que mejor paga en blogs.

**Regla de Oro:** Nunca ocultes contenido real detrás de un anuncio. Penalización SEO inmediata.

### Implementación Mobile (AdMob)
Para Apps React Native / Flutter.
1.  **Banner:** Barra inferior fija. Ingresos bajos pero constantes.
2.  **Intersticial:** Pantalla completa al cambiar de nivel/página. Ingresos altos, pero molesta al usuario. Úsalo con moderación (ej: cada 3 niveles).
3.  **Rewarded Video:** "Ver video para ganar una vida extra". El mejor balance UX/Ingresos.

---

## 2. Modelos SaaS (B2B / B2C)

### Freemium (El estándar)
*   **Free:** Funcionalidad básica limitada (ej: "Hasta 5 proyectos").
*   **Pro ($):** Ilimitado + Funciones Power User.

### Free Trial (Prueba Gratuita)
*   **Modelo:** Acceso total por 14 días. Luego paga o muere.
*   **Uso:** Mejor para B2B complejo donde el usuario necesita "sentir" el valor antes de pagar.

### Pay-per-use (Metered)
*   **Modelo:** Cobras por unidad (ej: "por cada 1000 emails enviados").
*   **Uso:** APIs, Infraestructura.

---

## 3. Marketing de Afiliados

Recomienda productos de otros y gana comisión.

1.  **Amazon Associates:** Para productos físicos.
2.  **SaaS Affiliates:** Recomienda hostings, herramientas de email (ej: ConvertKit, DigitalOcean). Pagan muy bien (20-30% recurrente).

**Estrategia:** Crea contenido de valor "Cómo montar un servidor" y pon tu link de afiliado de DigitalOcean.

---

## 4. Patrocinios y Lead Gen

Si tienes una audiencia de nicho muy específica, no uses AdSense. Vende el espacio directamente a una empresa de software jurídico o bufete grande.

*   **AdSense:** Puede pagarte $1 USD por click.
*   **Patrocinio Directo:** Puede pagarte $500 USD por un banner mensual.
