## Formula de Cobro

```
PRECIO = (Horas estimadas x Valor hora) x Factor de complejidad x Factor de urgencia

Factor complejidad:
  Simple:     1.0
  Media:      1.2
  Alta:       1.5
  Muy Alta:   1.8

Factor urgencia:
  Normal:     1.0
  Rapido:     1.3
  Urgente:    1.5 - 2.0
```

### Ejemplo Calculo

```
Proyecto: Ecommerce medio
Horas estimadas: 170h
Valor hora: $35.000 CLP
Complejidad: Media (1.2)
Urgencia: Normal (1.0)

PRECIO = 170 x $35.000 x 1.2 x 1.0 = $7.140.000 CLP
```

## Modelo de Cobro Sugerido

```
1. Anticipo: 40-50% al inicio del proyecto
2. Avance: 30% al entregar primera version funcional
3. Cierre: 20-30% al entregar version final

Para mantenimiento:
  - Plan basico: $30.000 CLP/mes (hosting + 2h soporte)
  - Plan medio:  $60.000 CLP/mes (hosting + 5h soporte + actualizaciones)
  - Plan premium: $120.000 CLP/mes (hosting + 10h soporte + desarrollo menor)
```
